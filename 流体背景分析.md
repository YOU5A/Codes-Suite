## 两个方案

| 方案 | 技术 | 复杂度 | 切入点 |

### A.3 React 组件 (可直接复制)

`	sx
// src/components/FluidBackgroundA.tsx
// ★ 方案 A: SVG + Canvas 2D 实现，零 WebGL 依赖
import { useRef, useEffect, useState } from "react";
import { FluidFilter } from "@/design-system/fluid/FluidFilter";

interface Props {
  albumUrl?: string;
  playing?: boolean;
  /** 位移强度 200-600，默认 400 */
  displacementScale?: number;
}

export function FluidBackgroundA({ albumUrl, playing = true, displacementScale = 400 }: Props) {
  const [c1, c2, c3, c4] = [useRef<HTMLCanvasElement>(null), useRef<HTMLCanvasElement>(null), useRef<HTMLCanvasElement>(null), useRef<HTMLCanvasElement>(null)];
  const containerRef = useRef<HTMLDivElement>(null);
  const feDisplacementMapRef = useRef<SVGFEDisplacementMapElement>(null);

  // 播放/暂停控制
  useEffect(() => {
    containerRef.current?.classList.toggle("paused", !playing);
  }, [playing]);

  // 位移强度控制
  useEffect(() => {
    feDisplacementMapRef.current?.setAttribute("scale", String(displacementScale));
  }, [displacementScale]);

  // 加载封面 → 四分块画到 canvas
  useEffect(() => {
    if (!albumUrl) return;
    const img = new Image();
    img.crossOrigin = "Anonymous";
    img.onload = () => {
      const { width, height } = img;
      const canvases = [c1, c2, c3, c4];
      const sx = [0, width / 2, 0, width / 2];
      const sy = [0, 0, height / 2, height / 2];
      const sw = width / 2, sh = height / 2;

      canvases.forEach((ref, i) => {
        const ctx = ref.current?.getContext("2d");
        if (ctx) {
          ctx.filter = "blur(5px)";
          ctx.drawImage(img, sx[i], sy[i], sw, sh, 0, 0, 100, 100);
        }
      });
    };
    img.src = albumUrl;
  }, [albumUrl]);

  return (
    <>
      <FluidFilter />
      <div className="rnp-background-fluid" style={{ backgroundImage: albumUrl ? url() : undefined }}>
        <div ref={containerRef} className="rnp-background-fluid-rect">
          {[c1, c2, c3, c4].map((ref, i) => (
            <canvas key={i} ref={ref} canvasID={String(i + 1)} width="100" height="100" />
          ))}
        </div>
      </div>
      {/* 更新 SVG filter 的 displacement scale */}
      <svg width="0" height="0">
        <filter id="fluid-filter">
          <feTurbulence type="fractalNoise" baseFrequency="0.005" numOctaves="1" seed={Math.floor(Math.random() * 1000)} />
          <feDisplacementMap ref={feDisplacementMapRef as any} in="SourceGraphic" scale={String(displacementScale)} />
        </filter>
      </svg>
    </>
  );
}
`

### A.4 优势

- ✅ 无 npm 依赖
- ✅ 无需 WebGL 知识
- ✅ Electron Chromium 原生支持 SVG Filter
- ✅ CSS animation-play-state 直接控制启停
- ✅ 可与现有 GlassBackground 共存

## 方案 B: AMLL WebGL Mesh Gradient (高品质)

### B.1 核心文件清单 (可复制)

| 源文件 | 行数 | 作用 |
|--------|------|------|
| ase.ts | ~100 | 基类 + ResizeObserver |
| enderer.ts (index.ts) | ~60 | Facade 包装器 |
| mesh.ts (mesh-renderer/index.ts) | ~1100 | WebGL 管线 + 动画循环 |
| mesh.frag.glsl | ~40 | 片段着色器 (纹理采样+dither+暗角) |
| mesh.vert.glsl | ~20 | 顶点着色器 (控制点位置+UV) |
| cp-generate.ts | ~200 | Perlin 噪声控制点生成 |
| cp-presets.ts | ~200 | 预设控制点数据集 |
| img.ts | ~140 | 高斯模糊+饱和度+对比度 |
| g-render.tsx (React) | ~130 | React 绑定 |

### B.2 依赖处理

AMLL 使用了以下 npm 包，Codes-Suite 可选择：

| 依赖 | 用途 | 替代方案 |
|------|------|----------|
| gl-matrix | Vec2/Vec3/Mat4 向量运算 | 自实现简单 Vec2 或安装 gl-matrix (MIT) |
| colorthief | 从封图提取调色板 | 可直接 
pm install colorthief (MIT) |

## 两方案对照

| 维度 | 方案 A (SVG) | 方案 B (WebGL) |
|------|-------------|----------------|
| 渲染引擎 | CSS SVG Filter | WebGL 1.0 |
| 实现难度 | 低 (纯 DOM) | 高 (GL管线) |
| 性能 | GPU (CSS filter) | GPU (手写shader) |
| 文件数 | 1 组件 + 1 CSS | 9 文件 |
| 代码量 | ~150 行 | ~2000 行 |
| 视觉效果 | 扭曲+旋转 | 网格变形+脉动 |
| 可定制性 | 仅 filter 参数 | 完整 shader 控制 |
| 建议用途 | **Phase 1 快速集成** | **Phase 2 高品质** |
